◆ bq27441_g1_bat_insert()

void bq27441_g1_bat_insert ( void  )

Forces the Flags() [BAT_DET] bit to set when the battery insertion detection is disabled via OpConfig [BIE] = 0. In this case, the gauge does not detect battery insertion from the BIN pin logic state, but relies on the BAT_INSERT host subcommand to indicate battery presence in the system.

Return values
